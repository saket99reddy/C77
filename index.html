to download visual studio code do the following steps.
1.Download the Visual Studio Code installer for Windows.
2.Once it is downloaded, run the installer (VSCodeUserSetup-{version}.exe). This will only take a minute.
3.By default, VS Code is installed under C:\Users\{Username}\AppData\Local\Programs\Microsoft VS Code .
(visual studio is free and can be used without internet.)
once you have done this to set it up you have to create a new file in file exporer then click on file in visual studio code
and click open folder and select the new file to get started.
then you have to click on the empty space below the name of the folder and click new file and then name it index.html and start coding
